# げんちゃんのアフター一本勝負 🥂

スリル満点！チャット型スナック・シミュレーター。
フィリピンパブのキャスト、ユンちゃんとの「駆け引き」を楽しみながらアフター（店外デート）を目指せ！

![ユンちゃん](image/yun_normal.png)

## 🎮 ゲーム概要
プレイヤーは「げんちゃん」となり、スナックでユンちゃんと会話します。
会話の内容に応じて、ユンちゃんがドリンクやフードをねだってきます。
財布の残高（¥15,000）を気にしながら、好感度を上げてアフターを勝ち取れるか？それとも破産するか、塩対応されるか……。

## ✨ 特徴
- **Gemini 3.1 Flash API によるリアルな会話**: カタコトの日本語と鋭い商売っ気が同居する、唯一無二の対話体験。
- **ダイナミック・おねだりシステム**: AIが会話の流れからメニューと価格を自由自在に決定。
- **自動課金システム**: チャットで「奢るよ！」と言っただけで、お財布からお金が引かれる（言ったもん負け）のスリル。
- **4つのエンディング**: 成功、破産、塩対応、そして……。

## 🛠️ セットアップ（開発者向け）
1.  リポジトリをクローン: `git clone https://github.com/ユーザー名/gen-quest2.git`
2.  `config.js.example` をコピーして `config.js` を作成。
3.  [Google AI Studio](https://aistudio.google.com/) でAPIキーを取得し、`config.js` に貼り付ける。
4.  `npx serve`（またはローカルサーバー）で起動。
